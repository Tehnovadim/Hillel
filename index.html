Команды терминала

  • cd - переместиться в указанную папку
  • pwd - вывести текущую папку
  • mkdir - создать папку
  • rmdir - удалить папку
  • touch - создать файл
  • cat - вывести содержимое файла
  • echo - вывести сообщение
  • ls - вывести список файлов и папок в текущей папке

Команды Git

  • git init - инициализировать репозиторий Git
  • git add - добавить изменения в репозиторий
  • git commit - создать коммит
  • git push - отправить изменения в удаленный репозиторий
  • git pull - загрузить изменения из удаленного репозитория
  • git branch - создать новую ветку
  • git checkout - переключиться на другую ветку
  • git merge - объединить две ветки
  • git log - просмотреть историю изменений
